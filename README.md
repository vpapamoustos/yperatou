# Yperatou - Performance Legends playable demo


## Δομή

- `index.html`
- `style.css`
- `script.js`
- `assets/cars/`

## Λειτουργικότητα

- Player vs Bot
- Player vs Player
- Κεντρικό home menu με ξεχωριστές οθόνες
- Local player profiles με επιλογή guest
- Edit/delete αποθηκευμένων προφίλ
- Local game history με pause/resume
- Φίλτρα ιστορικού ανά παίκτη, κατάσταση και deck
- Clear filters στο ιστορικό παιχνιδιών
- Auto-save ενεργών παιχνιδιών μετά από γύρους
- Quick Match με επιλογή 1-15 καρτών ανά παίκτη
- Classic Match 15 vs 15
- Time attack με επιλογή 1-20 λεπτών
- Επιλογή attribute
- Bot reveal
- Classic tie rule με pending pile
- Game over
